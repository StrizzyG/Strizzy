SQL and Databases Lessons and Exercises

Lesson 1: Introduction to SQL

SQL (Structured Query Language) is used to manage and manipulate databases.

Exercise: Write a SQL query to select all records from a table named "users".


SELECT * FROM users;

Lesson 2: SQL CRUD Operations

Learn about SQL CRUD operations: Create, Read, Update, and Delete.

Exercise: Write a SQL query to insert a new record into a table named "products".


INSERT INTO products (name, price, category) 
VALUES ('Product A', 50.00, 'Electronics');

Lesson 3: SQL Joins

Understand SQL joins to retrieve data from multiple tables.

Exercise: Write a SQL query to join "orders" and "customers" tables.


SELECT * 
FROM orders
JOIN customers ON orders.customer_id = customers.id;
